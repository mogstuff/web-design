Language Integrated Query (LINQ)

Introduction

LINQ is similar to SQL except it works on Collections. In plain English you can query pretty much anything that is in a List or bunch of lists.

Anatomy of a Query

A Query consists of the following stages:

  • Define the source.
  • Define query conditions
  • Get the filtered outpuut

IEnumberable

As per Microsoft Documentation the IEnumberable Interface is definded as an Interface which Exposes an enumerator, which supports a simple iteration over a non-generic collection.

The majority of LINQ queries will return a collection of type IEnumberable. In order to iterate through these you will need to use either ToList or ToArray.
