The only six fonts a designer needs

Massimo Vignelli

According to Massimo Vignelli a designer only needs these fonts.

There's a great video on youtbue covering this subject

  • Garamond
  • Bodoni
  • Helvetica
  • Century Expanded *
  • Futura *
  • Times Roman *
* not websafe or available via Google Fonts so need to be installed on computer or host the font file on your website.
